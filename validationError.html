ValidationError extends Error

constructor

(field: Field, reason: string)

The reason should contain a human-readable explanation for the validation error.

instance

reason: string

This contains a text description of the validation error.

field: Field

This contains the Field that the validation error occurred on.

message: string

This contains a longer text description of the validation error that takes both field and reason into account.

toString(): string

This returns message.
